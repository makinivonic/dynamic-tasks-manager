# 🚀 Dynamic Task Manager

A dynamic task management application built with vanilla JavaScript, featuring CRUD operations, DOM manipulation, and persistent local storage.

This project demonstrates practical frontend engineering skills by building an interactive and responsive task tracking system without external frameworks.

---

## 📖 Overview

This application allows users to create, update, delete, and manage tasks dynamically in the browser.

It focuses on core JavaScript concepts such as event handling, state management, DOM manipulation, and data persistence using local storage.

Each interaction updates the UI in real time, providing a smooth and responsive user experience.

---

## ✨ Features

- Create new tasks dynamically
- Mark tasks as complete or pending
- Edit existing tasks
- Delete tasks individually
- Persistent storage using localStorage
- Real-time UI updates without page reload
- Simple and responsive interface
